## Solana Token platform fees

These are fees paid to Solana Token on successful transactions. They are shown in the cost summary before you sign.

| Tool | Solana Token fee |
|------|-----------|
| Create Token | 0.1 SOL |
| Revoke Mint Authority | 0.05 SOL |
| Revoke Freeze Authority | 0.05 SOL |
| Revoke Update Authority | 0.05 SOL |
| Make Immutable | Free |
| Mint More Tokens | 0.05 SOL |
| Update Metadata | 0.05 SOL |
| Freeze / Unfreeze Account | 0.01 SOL per address |
| Create Pool (per DEX) | 0.1 SOL |
| Add / Remove Liquidity | 0.05 SOL each |
| Create OpenBook Market | 0.05 SOL |
| Multisender | 0.02 SOL per transaction |
| Burn Tokens | Free |
| Burn LP | Free |
| Analytics | Free |
| Security Checker | Free |

All fees are quoted in SOL and are subject to change. Always verify in the app before signing.

## Solana network costs (gas + rent)

These are paid to the Solana network, not to Solana Token.

| Cost type | Typical amount |
|-----------|----------------|
| Transaction gas | 0.0001–0.001 SOL |
| Token account rent (new account) | ~0.002 SOL |
| Mint account rent | ~0.002 SOL |
| Metadata account rent | ~0.015 SOL |

Rent is a deposit to keep on-chain accounts alive. In some cases it can be reclaimed by closing accounts.

## DEX pass-through costs

These are paid directly to the DEX protocol, not to Solana Token. Amounts are approximate and set by each DEX.

| DEX | Pool creation cost |
|-----|--------------------|
| Raydium CPMM | ~0.5 SOL |
| Raydium Legacy AMM | ~0.5 SOL (+ OpenBook market ~2.85 SOL) |
| Orca Whirlpools | ~0.5 SOL |
| Meteora DLMM | ~0.4 SOL |
| PumpSwap | ~0.4 SOL |
| Invariant | ~0.4 SOL |
| FluxBeam | ~0.4 SOL |

These figures are approximate. The exact cost is shown in the Solana Token UI before you sign. DEX costs can change without notice.

## Example: full token launch cost estimate

| Step | Cost |
|------|------|
| Create token | 0.1 SOL |
| Revoke mint authority | 0.05 SOL |
| Revoke freeze authority | 0.05 SOL |
| Make immutable | 0 SOL |
| Add Raydium CPMM pool (Solana Token fee) | 0.1 SOL |
| Raydium CPMM setup (pass-through) | ~0.5 SOL |
| Gas buffer | ~0.01 SOL |
| **Total overhead (excluding liquidity)** | **~0.81 SOL** |

Your liquidity deposit (the SOL and tokens you add to the pool) is separate and fully yours — it goes into the pool, not to Solana Token.

## Comparison with alternatives

Solana Token's all-in launch cost is approximately **0.1 SOL** for token creation, compared to alternatives that charge 0.3–0.5 SOL for the same action. Full launch with liquidity is approximately 0.81 SOL overhead vs ~1.3 SOL on comparable platforms.
