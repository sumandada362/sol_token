## What this tool does

Solana Token Market Create sets up an OpenBook v2 order-book market for your token on Solana. This creates a trading pair (your token vs SOL or USDC) and enables limit order trading — and satisfies the prerequisite for Raydium Legacy AMM pools.

## What it costs

| Item | Cost |
|------|------|
| Solana Token platform fee | 0.05 SOL |
| OpenBook rent deposit (pass-through) | ~2.85 SOL |
| Solana gas | ~0.001 SOL |

The ~2.85 SOL is a rent deposit paid directly to the OpenBook program — it is not a Solana Token fee. Total: approximately **2.9 SOL**. Costs can change — verify in the app.

## Before you start

- Your token already created (mint address ready)
- Approximately 3 SOL in your wallet
- Decision on min order size and tick size

<Warning>
**Think before creating a market.** If you are using Raydium CPMM (the modern default), you do not need an OpenBook market. Creating one costs ~2.85 SOL in rent. Only proceed if you specifically need Raydium Legacy AMM or want limit order support.
</Warning>

## Step-by-step

### Step 1 — Go to Solana Token → Tools → Create Market

Navigate to the Market Create tool.

### Step 2 — Enter your token mint address

Paste your token's mint address. Solana Token will auto-fill the token name and symbol.

### Step 3 — Choose base and quote currency

**Base currency:** Your token. **Quote currency:** SOL (default) or USDC. This defines the trading pair (e.g., MYTOKEN/SOL).

### Step 4 — Set market parameters

| Parameter | Description | Typical value |
|-----------|-------------|---------------|
| Min order size | Minimum amount tradeable | 1 (your token) |
| Tick size | Smallest price increment | 0.0001 SOL |

These parameters are permanent — they cannot be changed after market creation.

### Step 5 — Review cost and sign

The cost summary shows the Solana Token fee plus the OpenBook rent deposit. Click **Create market** and approve in your wallet.

### Step 6 — Save your market ID

After confirmation, copy your **Market ID** (a Solana public key). You will need this when creating a Raydium Legacy AMM pool.

## What happens on-chain

The transaction initializes several on-chain accounts in the OpenBook program: a market account, bids/asks order book, event queue, and request queue. The rent covers the storage cost for all of these accounts over the life of the market.
