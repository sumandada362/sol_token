## General questions

### Is Solana Token non-custodial?

Yes. Solana Token is a non-custodial tool. Your wallet signs every transaction. Solana Token never holds your private keys, seed phrase, or funds. We cannot access your wallet or reverse any transaction.

### What wallets does Solana Token support?

Phantom, Solflare, and Backpack. These are the most widely used Solana wallets. Additional wallet support may be added based on demand.

### Does Solana Token work on devnet?

Yes. Switch your wallet to devnet, fund it with free devnet SOL from a faucet (e.g., faucet.solana.com), and Solana Token will operate on devnet.

### Can I get a refund?

No. Platform fees paid on-chain are non-refundable. Solana transactions are irreversible by design. Always review the cost summary before signing.

## Token creation

### How much SOL do I need to create a token?

At minimum: **0.1 SOL** (Solana Token fee) + **~0.002 SOL** (Solana rent). If you also revoke mint authority (+0.05 SOL) and freeze authority (+0.05 SOL), budget **~0.20 SOL** plus gas.

### Is the token symbol permanent?

Yes. The symbol is stored in on-chain metadata and cannot be changed after creation. Triple-check it before signing.

### Can I create a Token-2022 token?

Yes. Choose Token-2022 in the token standard selector. Use this only if you specifically need an extension like transfer fees or confidential transfers. See the [Token-2022 docs](/docs/token-2022) for details.

### Can I test on devnet?

Yes. Switch your wallet to devnet and use a Solana devnet faucet. Solana Token automatically operates on the network your wallet is connected to.

## Authorities

### What is mint authority?

The right to create new tokens. If someone holds mint authority, they can increase your token's supply at any time. Revoking mint authority permanently caps the supply.

### What is freeze authority?

The right to freeze a token holder's account, preventing them from transferring tokens. Revoking freeze authority means no wallet can ever be frozen.

### What does "immutable" mean?

Immutable means update authority has been revoked. On-chain metadata (name, symbol, logo) can never be changed by anyone.

### Can I revoke authorities later?

Yes. You can revoke any authority at any time using Solana Token Tools, as long as you still hold that authority. Each revocation costs 0.05 SOL (except make immutable, which is free).

## Liquidity and DEXs

### Which DEX should I use?

Raydium CPMM is the most common choice for new token launches. It has lower setup costs, no OpenBook market required, and deep liquidity infrastructure. Orca and Meteora are solid alternatives.

### Do I need an OpenBook market?

Only if you are using Raydium Legacy AMM. Raydium CPMM does not require one. Most new tokens use CPMM.

### Can I add liquidity to multiple DEXs?

Yes. Solana Token lets you create pools on multiple DEXs in a single session.

## Safety and security

### Is Solana Token audited?

Solana Token uses established on-chain programs (Metaplex, SPL Token, Raydium, OpenBook). The Solana Token fee transfer logic is minimal. We recommend reviewing transactions in your wallet before signing.

### Can Solana Token rug pull my project?

No. Solana Token does not hold custody of your tokens, liquidity, or wallet. Once a pool is created, Solana Token has no access to it. Liquidity removal is controlled by whoever holds the LP tokens.

### How do I check if my token passes DEX scanner safety checks?

Use Solana Token Analytics or the Security Checker page. See also: [Token Authorities Explained](/docs/token-authorities).
