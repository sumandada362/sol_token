## What update authority is

Every Solana token created with Metaplex metadata has an **update authority** — the wallet allowed to change the token's on-chain metadata: name, symbol, and the metadata URI that points to the logo and description.

While the update authority is active, holders have no guarantee the token they bought keeps its identity. The authority holder could rename the token, swap the logo, or point the metadata at entirely different content.

## What revoking does

Solana's Metaplex metadata program has no concept of a "null" update authority. The standard way to revoke it is to **transfer the authority to the system program** (`11111111111111111111111111111111`) — an address that has no private key and can never sign a transaction.

After the transfer:

| Before | After |
|--------|-------|
| Your wallet can change name, symbol, and logo | No one can change anything |
| Scanners flag "update authority active" | Scanners show metadata as locked |
| Your wallet is listed as the token's authority | The token is no longer associated with your wallet |

## Revoke Update vs. Make Immutable

Solana Token offers two ways to lock metadata permanently:

| | Revoke Update | Make Immutable |
|---|---|---|
| Mechanism | Transfers authority to the system program | Sets the `isMutable` flag to `false` |
| Your wallet still listed as authority | No | Yes (but powerless) |
| Reversible | No | No |
| Cost | 0.05 SOL | Free |

Both achieve the same practical outcome — metadata can never change. Choose **Revoke Update** if you also want your wallet fully dissociated from the token's authority field; choose **Make Immutable** if you just want the lock.

<Warning>
**Revoking update authority is permanent.** Finalize your token's name, symbol, logo, and metadata URI before revoking. If anything needs fixing, use the Update Metadata tool first.
</Warning>

## How to revoke update authority

1. Open **Solana Token → Tools → Revoke Update**
2. Connect the wallet that currently holds the update authority
3. Paste your token's mint address — Solana Token verifies the authority on-chain
4. Check the irreversibility confirmation box and sign the transaction

Cost: **0.05 SOL** Solana Token fee + Solana gas (~0.000005 SOL).

## Verify on-chain

After the transaction confirms, search your mint address on Solscan. The "Update Authority" field shows the system program address (`1111…1111`), which scanners read as revoked.
